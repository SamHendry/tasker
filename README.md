# Tasker

This is a task CLI that I built for fun.

## Functions
Seperate each command, name, and property kwarg with a space. You can use indicies as names. The properties are: do, due, pri, and proj.

- **add_task:**
    - a \[names] \*\*\[properties]
- **remove_task:**
    - d \[names]
    - c \[names]
- **modify_task:**
    - m \[names] \*\*\[properties]
- **list_commands:**
    - help
